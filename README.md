# WokwiSim

Nesse repositório, estão alguns mini projetos desenvolvidos para se familiarizar com a plataforma Wokwi.

## Nano Blink

Essa é uma implementação de um LED Blink utilizando um Arduino Nano como plataforma. O código desse mini projeto é, essencialmente, o "Hello World" do mundo dos microcontroladores.

## Tiny Blink

Essa é uma implementação de um LED Blink utilizando um ATTiny85 como plataforma. O código utilizado é idêntico ao que fui utilizado no Arduino Nano.

## Tiny Semaphor Blink

Essa é uma implementação de um semáforo utilizando um ATTiny85 como plataforma. O código é muito semelhante às outras duas implementações, mas inclui uma função que controla os três LEDs simultaneamente de forma simplificada.
